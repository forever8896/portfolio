# Personal Portfolio
My personal portfolio website showcasing my projects and experience.

## Tech Stack
- SvelteKit
- TypeScript
- Tailwind CSS
- DaisyUI

## Development
# Install dependencies
```bash

npm install
```

# Start development server
```bash
npm run dev -- --open
```

## Deployment
The site is automatically deployed to [brianpistar.dev](https://brianpistar.dev) via Vercel.
